# githubavatar
Get Github Avatar
โปรแกรมสำหรับแสดงรูป avatar บน github ของผู้ใช้

## วิธีใช้งาน
- ติดตั้ง package ที่จำเป็นสำหรับโปรแกรม

```sh
pip install --user requests Pillow
```

## ติดตั้งโปรแกรม githubavatar 
```sh
pip install --user .
```

## วิธีการใช้งาน
```sh
githubavatar wichit2s
githubavatar joinshena
```
